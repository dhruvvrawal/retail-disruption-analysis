# D-Mart vs Blinkit — Indian Retail vs Quick Commerce Analysis

### Power BI Dashboard Project | Finance & Analytics

---

## Project Overview

This project analyzes whether quick commerce platforms like Blinkit are impacting traditional Indian retail businesses like D-Mart using financial, operational, and consumer trend data.

The dashboard focuses on:
- Revenue growth
- EBITDA margin trends
- Store expansion
- Consumer behavior
- Business insights

Built as a self-learning finance and analytics project using Power BI and Excel.

## Key Learnings

- Quick commerce is growing rapidly but traditional retail remains dominant in scale.
- D-Mart margins show early signs of pressure but long-term impact remains uncertain.
- Consumer behavior suggests both platforms currently coexist rather than fully replace each other.
